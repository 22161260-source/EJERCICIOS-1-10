# Ejercicios de Programación Web

Este repositorio contiene una serie de ejercicios desarrollados con HTML, CSS y JavaScript como parte del curso de Programación Web.

## Descripción de la actividad

Cada ejercicio se encuentra en su propio archivo HTML (y, cuando aplica, con sus respectivos archivos CSS y JS dentro de las carpetas `css/` y `js/`). El archivo `index.html`, ubicado en la raíz del repositorio, funciona como un menú principal con enlaces directos a cada uno de los ejercicios.

El ejercicio 10, por ejemplo, es una calculadora que convierte grados Celsius a Fahrenheit usando HTML y JavaScript, con validación de datos y un campo de resultado de solo lectura (`readonly`).

## Estructura del repositorio

```
├── index.html          -> Menú principal con enlaces a todos los ejercicios
├── ejercicio1.html
├── ejercicio2.html
├── ...
├── ejercicio10.html
├── css/
│   └── ejercicio10.css
└── js/
    └── ejercicio10.js
```
